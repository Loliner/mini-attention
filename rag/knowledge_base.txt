==============================
公司内部知识库（2026版）
==============================

【一、公司简介】

本公司成立于2018年，总部位于悉尼。
公司主要从事人工智能平台开发、数据中台建设以及企业级 SaaS 产品研发。
公司现有员工约200人。

--------------------------------

【二、年假政策】

1. 每位正式员工每年享有10天带薪年假。
2. 入职满1年后，额外增加3天奖励假。
3. 年假最多可以累计20天。
4. 离职时未使用的年假将按工资比例结算。
5. 实习生不享有带薪年假。

--------------------------------

【三、病假政策】

1. 每年5天带薪病假。
2. 连续请假超过2天需提供医院证明。
3. 重大疾病可申请特殊医疗假。

--------------------------------

【四、出差报销政策】

1. 国内出差每日住宿报销上限为200澳元。
2. 国际出差每日住宿报销上限为350澳元。
3. 机票必须选择经济舱。
4. 出差期间餐补为每日80澳元。
5. 所有报销必须在30天内提交。

--------------------------------

【五、远程办公政策】

1. 员工每周可申请2天远程办公。
2. 远程办公需提前1天申请。
3. 试用期员工不可申请远程办公。

--------------------------------

【六、IT安全规范】

1. 禁止在公共网络下访问公司数据库。
2. 所有员工必须开启双因素认证（2FA）。
3. 密码必须每90天更换一次。
4. 禁止使用个人邮箱发送公司机密文件。
5. 所有服务器日志保留180天。

--------------------------------

【七、技术架构说明】

公司核心系统采用微服务架构。
后端主要使用 Python 和 Go 语言开发。
数据库使用 MySQL 与 Redis。
消息队列采用 Kafka。
前端使用 React 技术栈。

--------------------------------

【八、绩效考核制度】

1. 每半年进行一次绩效评估。
2. 绩效等级分为 A、B、C、D 四档。
3. A 等级员工可获得20%年终奖金。
4. 连续两次 D 等级可能触发岗位调整。

--------------------------------

【九、员工福利】

1. 公司提供免费午餐。
2. 每年一次海外团建。
3. 提供心理健康咨询服务。
4. 每年提供1000澳元学习补贴。

--------------------------------

【十、设备申请流程】

1. 新员工可申请笔记本电脑一台。
2. 高级工程师可申请双显示器。
3. 设备损坏需提交 IT 工单。
4. 所有设备归公司所有。
